# Short-Message-Sender 

## 介绍
此程序为社团招新的通知发送设计，设计功能有：

1. 自定模板并自动替换变量
2. 批量发送短信 

## Dependence 

+ Python3.10
+ PySimpleGUI

## 使用方法

1. 工作表的处理
    1. 在工作表前添加两行
        ![示例一](/README_DEPENDENCE/1.jpg)
    2. 修改工作表首行的名称（必须要有的：电话号码（更名为PhoneNumber），可选模板号：Selnumber），其余的为变量名，在模板中定义
        ![示例二](/README_DEPENDENCE/2.jpg)
    3. 编写模板，模板分为常量和变量两部分
        + 常量

            在所有使用同一模板的信息内都一样的部分称之为常量，常量部分的编辑和普通信息一样，如：
            > xxx同学：<br>
             欢迎来到xx社团招新！期待在面试现场看到你的身影！<br>
             东园：<br>
             时间：9月24日（周六）8:30～12:00<br>
             地点：东教2栋305（常策部）<br>
             东教2栋306（表演部）<br>
            ️ 注意：<br>
             表演部面试请准备好一段3min左右的表演；面试现场会提供木吉他与小音箱，如果有特殊要求请自带乐器；如果不方便的话请提前录好视频并与我反映哦<br>
            
            
            这其中除开xxx的部分称为常量。
            由于excel限制，换行可用 __\n__ 代替，制表符可用 __\t__ 代替，普通的单斜杠请写两遍 __\\\\__
        
        + 变量

            变量是在信息中会变化的量，如时间，人名，录取声部等，在这里使用变量名代替，注意 __变量名前使用 *$* 标识，后面使用空格与正文隔开__
            
            比如：*$name $year* 等是变量名

            如果常量中出现美元符号 *$* ,使用 *\\\$* 代替
        
        + 示例

            > $name 同学，你好：\n恭喜你通过了我们合唱团的面试，正式成为了$part 声部的一员，请你在看到短信后添加声部长$name2 的微信，微信号是：$wechatid
            
            这个短信中的 *\$name \$name2 \$wechatid \$part* 将会被替换成你所需要的文字
        
    4. 将模板复制到第一行，每个单元格对应一个模板，如：A1对应1，B1对应2...,不同人所需要使用的模板在Selnumber列定义
    5. 打开手机，打开开发者选项、USB调试和指针位置选项(开发者选选项在设置关于系统点击版本号多次打开)
        ![USB调试](/README_DEPENDENCE/3.jpg "USB调试") ![指针位置](/README_DEPENDENCE/4.jpg "指针位置")
    6. 打开信息界面，找到发送按钮，点击并查看手指所在像素位置
        ![此处为所在像素位置](/README_DEPENDENCE/5.jpg "此处为所在像素位置")
    7. 在工作表A2中输入x值，B2输入y值
    ### 现在工作表可能是这样的：

    ![](/README_DEPENDENCE/6.jpg)
    
    8. 使用 *__.csv__* 格式保存表格
    9. 运行软件,在手机上同意调试的申请
    10. 使用软件打开保存的表格
    11. 等待，注意不要让手机息屏
